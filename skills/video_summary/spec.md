# 概要

動画ファイルから音声を抽出し、文字起こしを行った上で、その内容を要約するスキル。
長時間の動画の内容を素早く把握するため、文字起こしデータを中間ファイルとして出力・保存し、そのテキストデータをもとにLLMを用いてサマリー（要約）を生成する。

# 要件

- **入力**: 動画ファイル（.mp4等）のパスを受け取る。
- **音声抽出**: 動画ファイルから音声データ（.mp3等）を抽出する。
- **文字起こし**: 抽出した音声データから文字起こしを行う（OpenAI Whisper API等を想定）。
- **中間ファイル出力**: 文字起こし結果を中間ファイル（テキストまたはJSONフォーマット）として保存する。
- **要約生成**: 保存された文字起こしデータを読み込み、LLM（OpenAI GPT-4o-mini等）を用いて内容を要約する。
- **出力**: 最終的な要約結果をテキストとして出力・保存する。
- **APIキー管理**: OpenAI APIキーは環境変数から取得し、ハードコーディングしない。
- **疎結合設計**: 音声抽出、文字起こし、要約の各処理は独立したモジュールとして実装し、単独でもテスト・実行可能にする。

# 構成

- `src/video_summary/extractor.py`: 動画から音声を抽出するモジュール。
- `src/video_summary/transcriber.py`: 音声ファイルから文字起こしを行い、中間ファイルを出力するモジュール。
- `src/video_summary/summarizer.py`: 文字起こしデータを読み込み、要約を生成するモジュール。
- `src/video_summary/pipeline.py`: 上記モジュールを統合し、動画パスから要約までを一貫して実行するパイプライン。
- `tests/`: 各モジュールの正常系・異常系テストを配置。
